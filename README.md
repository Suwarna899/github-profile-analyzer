# 🚀 GitHub Profile Analyzer

A modern web-based dashboard that fetches and displays real-time GitHub user statistics using the GitHub REST API. It visualizes developer profiles in a clean and interactive UI with charts and insights.

---

## 📌 Features

- 🔍 Search any GitHub user
- 👤 View profile details (avatar, bio, location)
- 📦 Display total repositories, followers, and following
- ⭐ Show top repositories based on stars
- 📊 Visualize programming languages using charts
- ⚡ Real-time data using GitHub API
- 🎨 Modern dark-themed UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- GitHub REST API
- Chart.js
